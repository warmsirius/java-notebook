# 3.赋值运算符

## 3.1 赋值运算分类
* 符号: = 
    * 当 "=" 两侧数据类型不一致时，可以使用自动类型转换或使用强转类型转换原则进行处理。
    * 支持连续赋值

* 扩展赋值运算符
    * += 
    * -=
    * *=
    * /=
    * %=


## 3.2 赋值运算注意事项
* 扩展赋值运算不会改变变量本身数据类型
* 开发中，如果希望变量实现+2的操作(前提: int num = 10)，有2种方法: 
    * 方式一: num = num + 2;
    * 方式二: num += 2;(推荐)
    > `注意`: (num++)++是不对的。只有上面两种。
* 开发中，如果希望变量实现+1的操作(前提: int num = 10)，有3种方法:
   * 方式一: num = num + 1;
   * 方式二: num += 1;
   * 方式三: ++num; num++(推荐)
