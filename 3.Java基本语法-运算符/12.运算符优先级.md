# 12. 运算符优先级

运算符有不同优先级，所谓优先级就是表达式中的运算顺序。

| 顺序 | 运算符 |
|----|----|
| R-->L | . () {} ; , ++ -- ~ ! (data type) |
| L-->R | * / % |
| L-->R | + - |
| L-->R | << >> >>> |
| L-->R | < > <= >= instanceof |
| L-->R | == != |
| L-->R | & |
| L-->R | ^ |
| L-->R | \ |
| L-->R | && |
| L-->R | \\ |
| R-->L | ? : |
| R-->L | = *= /= %= += -= <<= >>= >>>= &= ^= \= |


`注意`: 只有单元运算、三元运算、赋值运算是从右向左的。
