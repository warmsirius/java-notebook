# 5. 比较运算符

## 5.1 比较运算符概述

| 运算符 | 运算 | 范例 | 结果 |
|----|----|----|----|
| == | 相等于 | 4==3 | false |
| != | 不等于 | 4!=3 | true |
| < | 小于 | 4<3 | false |
| \> | 大于 | 4>3 | true |
| <= | 小于等于 | 4<=3 | false |
| \>= | 大于等于 | 4>=3 | true |
| instanceof | 检查是否是类的对象 | "Hello" instanceof String | true |


比较运算符的结果都是boolean型，要么为true，要么是false。


## 5.1 比较运算符注意
* 比较运算“==”不能写成“=”。
