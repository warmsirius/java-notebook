# 15. 嵌套循环练习

1.打印并输出九九乘法表
```java
class NestedCycleExercise {
    public static void main(String[] args) {
        //1.打印99乘法表
        for (int i = 1; i <= 9; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print(i + " * " + j + " = " + i * j + "  ");
            }
            System.out.println();
        }
    }
}
```

2.100以内所有质数

```java
class NestedCycleExercise {
    public static void main(String[] args) {
        //2.100以内所有的质数
        for(int i = 2; i <100; i++){
            boolean isPrimeNumber = true;
            for(int j = 2; j < i; j++){
                if (i % j == 0){
                    isPrimeNumber = false;
                    break;
                }
            }
            if(isPrimeNumber){
                System.out.println(i);
            }
        }
    }
}
```
