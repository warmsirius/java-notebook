# 2. if-else

## 2.1 if-else格式
条件判断结构。


### 格式1: 单独if
```
if(条件表达式){
    执行代码块;
}
```

![if-else格式1](../4.Java基本语法-流程控制-assets/ifelse-1.jpg)


### 格式2: if-else
```
if(条件表达式){
    执行代码块1;
}
else{
    执行代码块2;
}
```
![if-else格式2](../4.Java基本语法-流程控制-assets/ifelse-2.jpg)


### 格式3: if-else if-else

```
if(条件表达式1){
    执行代码块1;
}
else if(条件表达式2){
    执行代码块2;
}
...
else{
    执行代码n;
}
```

![if-else格式2](../4.Java基本语法-流程控制-assets/ifelse-3.jpg)
